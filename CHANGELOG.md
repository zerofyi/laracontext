# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-20

### Added

- Initial release of LaraContext.
- `context:generate` artisan command to compile a structured Markdown context file for AI assistants.
- Schema inspection, model relationship mapping, and `FormRequest` validation extraction.
- `--output` and `--no-routes` flags.
